# Connect Application
### This is a backend of social media application.

<br>

## Technologies
- **Language** - Node.js (Javascript)
- **Framework** - Express4
- **Database** - MySQL & REDIS
- **Storage** - Multer (Disk storage)

<br>

## Third-party services
- **Emails** - Nodemailer & Google api

<br>

## Design pattern
- **MVC** (Model View Controller)
